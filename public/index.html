NETHub

A web app designed for sharing files while on the same network. Especially useful when you are trying to get a file from a friend and their device has a single broken USB port.

  • Sharing made easy with QR Code.
  • Easy to use Drag and Drop file upload.
  • Faster than uploading to a server then downloading since you are the server.
  • Works with large files (tested with >2gb).
  • Quick response time.
